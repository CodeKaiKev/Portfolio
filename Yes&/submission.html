Summary

Innovation & Aspire is an ambitous new project which aims to make every emplopyee in an organisation feel heard and valued. Through a virtual suggestion box users are able to submit ideas/suggestions about the company to help them improve further. Following through on this, other users are able to reply to these suggestions through "Yes and...", this helps create a collaboration and helpful environment.
